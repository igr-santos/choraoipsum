Chorao Ipsum
============

Um gerador verbatim com um jeito sk8 de ser


Skate santos
------------

Instale o pacote:

`pip install choraoipsum`

Ou

`git clone https://github.com/igr-santos/choraoipsum.git`

`cd choraoipsum && pip install -r requirements.txt`

Gerar choratim:

`python -m choraoipsum --count 100`
